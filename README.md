## Requirements
- [esx_license](https://github.com/ESX-Org/esx_license)

## Download & Installation

### Using Git
```
cd resources
git clone https://github.com/PanScoobiik/esx_dmvschool
```

## Installation
- Import `esx_dmvschool.sql` in your database
- Add this in your `server.cfg`:

```
start esx_dmvschool
```
